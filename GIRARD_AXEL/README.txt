Bonjour,

Comme vous le demandiez voici mon niveau en web : des bases en d'HTML, CSS et php

Pour lancer le projet il suffit d'ouvrir app.html dans un navigateur récent.
Ensuite pour toute les actions qu'il est possible de faire tout ce faire dans le menu. Le menu est fixe et s'ouvrira à chaque fois que le curseur sera deçu.
Dans ce menu vous aurez la possibilité de changer la taille des images de preview, de faire un tri sur les images et enfin de faire une recherche sur l'api de reddit.

Dans le fichier "app.html" il y a une erreur car dans le tableau il y a une ligne <tr> mais pas de données <td> car elles sont écrite dans le javascript.
